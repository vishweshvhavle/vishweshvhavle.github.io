4DHumans: Temporally Consistent Dynamic Human Capture

Kunwar Maheep Singh, Astitva Srivastava, and Avinash Sharma

About

The 4DHumans dataset aims to capture multiple temporally consistent sequences of humans performing different actions. The sequences are approximately one minute long and captured using a multi-view setup of calibrated Microsoft Kinect cameras.

Each sequence is post-processed using the pipeline proposed in Fusion4D. The dataset will be made publicly available soon.
